### 📦 Structure Données

#### Lot Object
```javascript
{
  number: "LOT-2025-001",           // ID unique
  quantity: 10,                      // Quantité actuelle
  expiryDate: "2026-12-31",         // Date expiration
  dateAdded: "2025-01-15T10:00:00Z", // Quand créé
  lastUpdated: "2025-01-15T10:00:00Z" // Dernière modif
}
```

#### Stock avec Lots
```javascript
{
  id: 3001,
  nom: "Encre Noire",
  quantite: 8,           // TOTAL auto-recalculé
  batches: [             // Array lots
    { number: "LOT-001", quantity: 5, ... },
    { number: "LOT-002", quantity: 3, ... }
  ]
}

// Quantité = LOT-001(5) + LOT-002(3) = 8 ✓
```

### 🔄 Auto-Calcul Quantité

```javascript
// AVANT (manually)
Quantité = 8

// APRÈS (auto from lots)
Lot 1: 5
Lot 2: 3
+ Lot 3: 2
─────────
Total: 10 ✓ (auto-updated)
```

**Avantage** : Quantité = somme lots (jamais d'écart)
